## 系统简介

### 系统概述

本系统是一个基于 Spring Boot + Vue 3 的企业级权限管理系统，提供完整的用户、角色、权限、部门管理功能，支持数据权限控制、操作日志记录、系统监控、股票数据分析、知识库管理、消息推送等功能。

### 技术架构

**后端技术栈：**
- Spring Boot 2.7.x
- Spring Security + JWT
- MyBatis Plus
- Redis
- MySQL 8.0
- Knife4j API 文档

**前端技术栈：**
- Vue 3 + Vite
- Element Plus
- Pinia 状态管理
- ECharts 图表
- Axios HTTP 客户端

### 系统特点

- 🔐 完善的权限管理体系（RBAC模型）
- 📊 灵活的数据权限控制
- 📝 详细的操作日志记录
- 🔔 实时通知公告功能
- 📈 系统监控与缓存管理
- 📉 股票数据分析与智能推荐
- 📚 知识库文档管理
- 💬 消息订阅与多渠道推送
- 🤖 AI 大模型集成
- 🎨 美观的UI界面

## 数据权限说明

系统支持五种数据权限范围：

| 范围 | 说明 |
|------|------|
| 全部数据 | 可以查看所有数据 |
| 本部门及以下 | 可以查看本部门及下级部门数据 |
| 本部门 | 只能查看本部门数据 |
| 仅本人 | 只能查看自己创建的数据 |
| 自定义 | 自定义数据权限规则 |

## 常见问题

### 1. 登录相关

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 账号被锁定怎么办？**
A: 登录失败次数超过限制（默认5次），账号会被锁定30分钟。

### 2. 权限相关

**Q: 为什么看不到某些菜单？**
A: 菜单显示取决于用户角色权限，请联系管理员分配权限。

**Q: 如何给用户分配权限？**
A: 在"角色管理"中创建角色并分配权限，然后在"用户管理"中为用户分配角色。

### 3. 股票推荐相关

**Q: 股票推荐是如何计算的？**
A: 系统基于多规则评分引擎，综合技术面和基本面指标计算股票得分。

**Q: 如何配置推荐规则？**
A: 在"规则配置"中可调整各规则的权重和参数。

### 4. 消息订阅相关

**Q: 如何接收基金估值推送？**
A: 在"消息订阅"中开启基金估值推送，并选择推送渠道。
